PROJECT1: BUILD A GAME 

MEMORY GAME:

The game involves flipping face-down cards on a grid with a hidden image.
The player selects two cards to flip. If the images on the two flipped cards match,
The player finds a pair. If not, the player has to try again.

The game continues until all cards are face-up, with the goal being to match all pairs.
in as few moves as possible. When the player starts playing, the chronometer starts counting.
Once you have finished flipping all the cards, the chromometer will automatically stop.
and the player will see how long it took.
Once all pairs are matched, a "Congratulations! You win. Refresh to play again" message is displayed.
